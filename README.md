# DI725 Assignment 1 - Sentiment Analysis

This project focuses on sentiment classification of customer service conversations using transformer-based models.

## Steps
- Exploratory Data Analysis (EDA)
- Data preprocessing and stratified splitting
- Transformer-based model (DistilBERT)
- Evaluation using macro F1-score

## Tools
- PyTorch
- HuggingFace Transformers
- Weights & Biases
